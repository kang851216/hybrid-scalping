# 5D Hybrid Scalping Strategy — 완벽 가이드

> **핵심 원칙**: 5일 수익률이 양수면 바이앤홀드, 음수면 RSI(2) 스캘핑

---

## 1. 전략 개요

### 1.1 한 줄 요약

```
close > close[5일 전]  →  전량 매수 후 홀딩 (Buy & Hold)
close ≤ close[5일 전]  →  RSI(2) < 25일 때 Kelly 20% 매수 후 SL/TP 브래킷
```

### 1.2 백테스트 성과

| 테스트 | 횟수 | 승률 | 평균 수익률 |
|--------|------|------|-----------|
| 미국 주식 2025~2026 | 23회 | 96% | +61% |
| 미국 주식 2020~2024 | 7회 | 100% | +34% |
| 한국 주식 2026 | 2회 | 50% | +49% |
| Top5 포트폴리오 | 12회 | 92% | +33% |
| **전체** | **89회** | **89%** | **+45%** |

---

## 2. 진입 규칙

### 2.1 스캘핑 모드 (5D Return ≤ 0)

```
RSI(2) < 25  →  LONG 진입 (매수)
RSI(2) ≥ 25  →  관망

※ SHORT(공매도)는 사용하지 않음 (LONG only)
```

### 2.2 바이앤홀드 모드 (5D Return > 0)

```
5일 수익률이 양수로 전환  →  모든 스캘핑 포지션 청소 + 전량 매수
5일 수익률이 음수로 전환  →  보유 주식 전량 매도 + 스캘핑 모드 복귀
```

---

## 3. 리스크 관리

### 3.1 ATR 동적 SL/TP

```python
ATR = 14기간 평균 진폭
SL  = clip(ATR% × 0.5,  min=0.1%,  max=0.8%)
TP  = SL × 10  (10:1 손익비)
```

| 종목 | ATR% | SL | TP |
|------|------|-----|-----|
| AMZN | 0.92% | 0.46% | 4.6% |
| NAVER | 0.97% | 0.49% | 4.9% |
| C | 0.69% | 0.35% | 3.5% |

### 3.2 Kelly 포지션 사이징

```python
f = (p × R - q) / R × 0.5
  = (0.20 × 10 - 0.80) / 10 × 0.5
  = 자산의 10%
```

> Kelly 100% = 12%, 보수적으로 50% 할인 → 6~10%

### 3.3 트레일링 스탑

```
이익이 TP의 50% 도달  →  SL을 본전(매수가)으로 이동
→ 이후 하락해도 손실 없이 청산
```

### 3.4 VIX 필터

| VIX | 동작 |
|-----|------|
| < 20 | 정상 운영 |
| 20~25 | 경계 (포지션 유지) |
| 25~30 | 포지션 50% 축소 |
| > 30 | **전체 거래 중단** |

### 3.5 손실 한도

| 레벨 | 한도 | 동작 |
|------|------|------|
| 일일 | -3% | 해당 종목 거래 중단 |
| 주간 | -8% | 전체 거래 중단 |

### 3.6 시간 필터

```
장 시작 30분 (9:30~10:00)  →  진입 금지 (스프레드 넓음)
장 마감 30분 (15:30~16:00) →  진입 금지 (변동성 과다)
```

## 4. 종목 선정 (주간 스크리너)

### 4.1 필터 조건

| 조건 | 기준 | 이유 |
|------|------|------|
| ADX | < 25 | 박스권 종목만 (추세는 제외) |
| ATR% | > 0.4% | TP 도달 가능한 충분한 변동성 |
| 가격 | $20 ~ $600 | SL이 의미 있는 구간 |
| 갭 리스크 | < 12% | SL 무력화 방지 |

### 4.2 점수 산정

```python
score = ADX_score(30%) + Vol_score(25%) + Gap_score(15%) + Price_score(15%) + Trend_score(15%)

ADX_score   = max(0, 25-ADX)/25 × 30   # 낮을수록 좋음
Vol_score   = min(ATR%/1.0, 1) × 25     # 높을수록 좋음
Gap_score   = max(0, 12-MaxGap)/12 × 15 # 낮을수록 좋음
Price_score = (1 - |price-200|/400) × 15 # $200 근처 최적
Trend_score = max(0, (ret30d+15)/30) × 15 # 하락 덜한 것
```

### 4.3 운영

```
매주 토요일: 스크리너 실행 → 상위 종목 모니터링 (참고용)
분기 1회:   종목 교체 (3개월마다 — 너무 자주 바꾸면 LLN 미발동)
종목당:     $1,000 ~ $5,000 배정

⚠️ 주간 교체 절대 금지!
  → 주간 교체 시 종목당 거래 2~12회 → LLN 발동 불가 → 수익률 -28% (검증 완료)
  → 분기 교체 시 종목당 거래 100~150회 → LLN 완전 발동 → 수익률 +28% (검증 완료)
```

## 6. 전체 알고리즘

```
매일 아침 9:30:
│
├─ 1. VIX 체크
│   ├─ > 30 → 오늘 거래 중단
│   └─ > 25 → 포지션 50% 축소
│
├─ 2. 5D Return 체크
│   ├─ > 0 → BH 모드 (전량 매수/홀딩)
│   └─ ≤ 0 → 스캘핑 모드
│
├─ 3. 스캘핑 모드 (매 시간/5분마다)
│   ├─ 시간 필터 (10:00~15:30만 거래)
│   ├─ RSI(2) < 25 → LONG 진입
│   ├─ SL = ATR% × 0.5 (0.1~0.8%)
│   ├─ TP = SL × 10
│   ├─ Kelly 10% 포지션
│   ├─ 트레일링 스탑 (이익 50% → 본전)
│   └─ 일일 -3% → 거래 중단
│
└─ 4. 매주 토요일: 스크리너 → 모니터링
   매 분기 1회: 종목 교체 (1월, 4월, 7월, 10월 첫날)
```
